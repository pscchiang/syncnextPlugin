# syncnextPlugin (自用)

一些自用的小插件，不保證可用，隨緣更新。

PS.目前鬥魚的高碼率源只提供flv格式，syncnext 使用 avplayer 暫時不支持flv格式，所以目前僅提供較低碼率的手機源。
